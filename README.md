# Mammo Second Reader

An auditable deep-learning pipeline for breast cancer screening in mammography. The system assists radiologists by providing a second opinion on mammogram reads, with reproducible preprocessing and evaluation workflows.

This is my Git repository for CM3070 Computer Science Final Project.

## Disclaimer

This software is intended for **research use only** and is not validated for clinical decision-making.
